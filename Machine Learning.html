Machine Learning

The following code demonstrates two distinct end-to-end machine learning pipelines, designed to handle different types of data: structured data (tabular) and unstructured data (images). Each pipeline addresses the respective problem with two models: a classic machine learning model (Random Forest) and a deep learning model (CNN for images and MLP for tabular data). The goal is to preprocess, train, and export models for both data types, providing a comprehensive approach for handling diverse machine learning tasks.
